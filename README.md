# 🔐 Password Strength Checker

![Python](https://img.shields.io/badge/Python-3.13-blue?logo=python)
![CustomTkinter](https://img.shields.io/badge/GUI-CustomTkinter-green)
![Status](https://img.shields.io/badge/Project-Completed-success)
![License](https://img.shields.io/badge/License-Educational-orange)

A modern Python GUI application that analyzes password strength using cybersecurity best practices. The application evaluates passwords based on multiple security rules and provides real-time feedback to help users create stronger passwords.

---

## ✨ Features

- 🔐 Password Strength Analysis
- ⚡ Real-Time Password Checking
- 👁️ Show / Hide Password
- 📋 Copy Password
- 📊 Password Strength Meter
- 🟢 Weak / Medium / Strong Detection
- ✔️ Live Password Requirement Checklist
- 🚫 Common Password Detection
- 💡 Password Improvement Suggestions
- ⏳ Estimated Password Crack Time
- 🎨 Modern Dark Theme GUI

---
